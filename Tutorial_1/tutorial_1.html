Sublime text

Sublime text-проприетарный текстовый редактор, который поддерживает большое количество языков программирования. Автором является человек по имени Джон Скиннер

Visual Studio Code

Visual Studio Code- редактор исходного кода, который позиционирует себя как "лёгкий", разработанный Microsoft для Windows, Linux и macOS.

Eclipse

Eclipse- свободная интегрированная среда разработки модульных кроссплатформенных приложений. Развивается и поддерживается Eclipse Foundation.
